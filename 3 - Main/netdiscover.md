## Usage
```
netdiscover -r 192.168.0.0/24
```

| IP          | MAC               | Count | Len | MAC Vendor / Hostname |
| ----------- | ----------------- | ----- | --- | --------------------- |
| 192.168.0.1 | 08:00:27:33:d3:f1 | 4     | 240 | ZTE Corporation       |
> [!tip] MAC addresses starting with 08:00:27 are VirtualBox ones
